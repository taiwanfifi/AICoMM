三、Entropy 是什麼？為什麼說理論矛盾？
好，這個最關鍵，我用超簡單版本。
Entropy 在這裡代表：
模型的注意力分布「有多分散」。
🟢 低 entropy（很集中）
假設模型對 10 個 token 分配注意力：
[0.90, 0.01, 0.01, 0.01, ...]
幾乎 90% 注意力都在一個 token 上。
→ 很集中
→ entropy 低
🔴 高 entropy（很分散）
[0.10, 0.10, 0.10, 0.10, 0.10, 0.10, 0.10, 0.10, 0.10, 0.10]
每個 token 差不多。
→ 很分散
→ entropy 高
你的數據是：
模型	Entropy
3B	4.21
14B	4.65
7B	5.49
數值越大 = 越分散。
所以：
3B 最集中
14B 中等
7B 最分散
但論文說什麼？
論文說：
7B 因為比較小 → 注意力比較集中 → 所以選 token 比 14B 好
這裡就出問題了。
因為數據顯示：
7B 是最分散的，不是最集中。
這就叫：
👉 internal inconsistency
意思是：
論文裡的解釋，跟論文裡自己的數據互相矛盾。
不是 reviewer 攻擊
是你自己 paper A 打 paper B。
這是很危險的。
